# MoodVault 🎬

MoodVault is a mood-based anime and movie discovery + personal vault app.

## Tech Stack

- React (Vite)
- Firebase Auth
- Firestore
- React Router
- TanStack Query
- Tailwind CSS

## Features (V1)

- Authentication
- Explore movies & anime
- Personal watch vault
- Mood-based suggestions
- Offline-friendly caching
